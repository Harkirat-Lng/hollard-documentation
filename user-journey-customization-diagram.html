📋 Feature Details by Tab (Click to expand)

📊 Overview Tab ▼

  • Quick access to create new User Journey
  • Quick access to create new Leaderboard Banner
  • View all active User Journeys in accordion
  • View all active Banners in accordion
  • Edit existing User Journeys
  • Delete User Journeys (moves to recycle bin)
  • Edit existing Banners
  • Preview Banners before publishing
  • Delete Banners (moves to recycle bin)

🛤️ Manage User Journeys Tab ▼

  • Set User Journey Name (internal identification)
  • Define Campaign UTM ID (for tracking URLs)
  • Configure Begin Date with date picker
  • Configure End Date with date picker
  • Option for "No End Date" campaigns
  • Select Campaign Category from dropdown
  • Customize Landing Page (future feature)
  • Customize Login Page (future feature)
  • Choose: No Banner / Existing Banner / New Banner
  • Save and Publish journey

🏆 Manage Leaderboard Banner Tab ▼

  • Set Banner Name (internal identification)
  • Configure Banner Heading (visible to users)
  • Rich text Banner Content with CKEditor
  • Set Button Text (CTA text)
  • Configure Button Action:
  • • Redirect to on-platform page
  • • Redirect to off-platform URL
  • • Link to specific Product Tile
  • Preview banner before publishing
  • Save and Publish banner

🗑️ Recycle Bin Tab ▼

  • View deleted/inactive User Journeys
  • View ended campaigns
  • View deleted/inactive Leaderboard Banners
  • Review past campaign details
  • Historical record keeping
